# TEFFKO

This is a Java based Test-Framework for KONNEX (KNX) based konfigurations. Its main purpose is to create automated tests for existing KNX-setups in a xUnit kind of style. This Framwork uses parts of the [Calimero Project](https://calimero-project.github.io/). Please note that this is not a commercial approach but a project for a bachelors thesis. Feel free to clone and fork this project.
